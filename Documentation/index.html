Getting started

Welcome

Hello there! Thanks for buying NPanel. This is a Login/Register script(template) made with PHP, MySQL, HTML, CSS, Javascript, jQuery and other modern jQuery plugins. This script(template) has made especially for new coders to make it easier with starting a site.

This template uses Twiter's Bootstrap Framework and custom CSS.

Folder structure

Once you download the zip file of npanel, unzip it. You should have a folder named NPanel with the following folders.

Folder Contents
Documentation Documentation and help instructions.
NPanel NPanel Script file.

NPanel Script file contain these folders and files.

Folder Contents
classes Core files for hashing password and PHP classes also containt and PHPmailer for verification and reset mail.
css All CSS style used in NPanel.
fonts All files used in NPanel.
images All images used in NPanel.
includes Contain a config file required to connect with database and many others.
js All javascript files used in NPanel.
vendor All files used in NPanel.
activate.php File to activate user after register.
dashboard.php Homepage of NPanel.
index.php Index page/Login page of NPanel.
logout.php Required to process logout user.
reset.php Page to require new password.
resetpassword.php Confirm password(last step).
signup.php Register page.

Installation

Install the template via ftp :

  1. Connect to your website host, using some of many FTP clients, for example FileZilla.
  2. Go to your default server folder, and copy NPanel files into that folder.
  3. Create MySQL database and replace all info with yours data.
  4. Import inside database our DB file(it's located inside NPanel folder with name DB.sql.

Customizing NPanel

Changing database connection

You can simply change database info in a few steps: Inside folder INCLUDES->CONFIG.php 

	
		<define('DBHOST',' '); //here put your host, probably localhost
                define('DBUSER',' '); //your MySQL username
                define('DBPASS',' '); //your MySQL password
                define('DBNAME',' '); //your MySQL database name>

Changing PHPMailer info

To change SMTP mail info, go to CLASSES->PHPMAILER->MAIL.php

          	

              < public $From     = ' '; //your email here
                public $FromName = SITETITLE;
                public $Host     = ' '; //your SMTP server (you can use smtp.gmail.com service)
                public $Mailer   = 'smtp';
                public $SMTPAuth = true;
                public $Username = ' '; //your email to send mail
                public $Password = ' '; //your password of email
                public $SMTPSecure = 'tls'; //protocol TLS or SSL
                public $WordWrap = 75;<
